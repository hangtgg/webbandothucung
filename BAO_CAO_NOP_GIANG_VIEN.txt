1.1. Lý do chọn đề tài
-----------------------
  Thị trường thú cưng Việt Nam đang trên đà tăng trưởng mạnh mẽ với tốc độ
  tăng trưởng ước tính 15%/năm. Nhu cầu mua sắm trực tuyến các sản phẩm và
  dịch vụ chăm sóc thú cưng ngày càng cao, đặc biệt trong giới trẻ tại các
  đô thị lớn.

  Nhóm lựa chọn xây dựng website thương mại điện tử chuyên về thú cưng với
  mục tiêu ứng dụng các công nghệ hiện đại (AI Chatbot, Voice Search, Image
  Search) để mang lại trải nghiệm mua sắm thông minh và tiện lợi cho người dùng.

1.2. Mục tiêu đề tài
---------------------
  - Xây dựng hệ thống website TMĐT thú cưng hoàn chỉnh với kiến trúc
    Client-Server rõ ràng.
  - Cơ sở dữ liệu 70+ sản phẩm, phân bố trên 7 danh mục thú cưng đa dạng.
  - Tích hợp Chatbot AI hỗ trợ tư vấn sản phẩm bằng tiếng Việt (NLP pattern).
  - Triển khai tính năng tìm kiếm đa phương thức: văn bản, giọng nói, hình ảnh.
  - Xây dựng hệ thống gợi ý sản phẩm thông minh (Content-Based Filtering).
  - Giao diện responsive, thân thiện, hoạt động tốt trên mọi thiết bị.

1.3. Phạm vi đề tài
--------------------
  - Đối tượng hướng đến: Người nuôi thú cưng (chó, mèo, hamster, thỏ,
    chim, cá, bò sát) tại Việt Nam.
  - Phạm vi địa lý: Không giới hạn (website trực tuyến).
  - Công nghệ: Node.js (Backend), HTML/CSS/JavaScript thuần (Frontend).

PHẦN V. TÍNH NĂNG CHI TIẾT VÀ THUẬT TOÁN
================================================================================

5.1. Chatbot Tư vấn AI
-----------------------
  Chatbot sử dụng phương pháp NLP Pattern Matching đơn giản:
    1. Người dùng nhập câu hỏi (tiếng Việt)
    2. Server nhận POST /api/chatbot/message
    3. Phân tích từ khóa trong message: nhận diện intent
    4. Trả về response phù hợp + danh sách sản phẩm gợi ý

  Ví dụ intent được nhận diện:
    - "thức ăn", "ăn", "đói"  → Gợi ý nhóm thức ăn
    - "vệ sinh", "cát", "bồn" → Gợi ý nhóm vệ sinh
    - "đồ chơi", "chơi"       → Gợi ý đồ chơi
    - "bệnh", "ốm", "khám"    → Tư vấn thú y

5.2. Hệ thống Gợi ý Sản phẩm (Content-Based Filtering)
--------------------------------------------------------
  Thuật toán tính điểm tương đồng:

    Điểm(A, B) = tổng [ trọng_số_i × khớp(A, B, tiêu_chí_i) ]

    Trong đó:
      • relatedProducts khớp trực tiếp   : +3 điểm
      • Cùng category (cat/dog/general)  : +2 điểm
      • Cùng tags (từ khóa đặc trưng)    : +1 điểm/tag

  Kết quả: Top 5 sản phẩm có điểm cao nhất được trả về.

5.3. Tìm kiếm Đa phương thức
------------------------------
  a) Tìm kiếm văn bản:
     - GET /api/search/text?q={keyword}
     - So khớp tên, mô tả, tags sản phẩm (case-insensitive)

  b) Tìm kiếm giọng nói:
     - Sử dụng Web Speech API (SpeechRecognition)
     - Ngôn ngữ: vi-VN (tiếng Việt)
     - Chuyển audio → text → gọi API tìm kiếm văn bản

  c) Tìm kiếm hình ảnh:
     - POST /api/search/image (multipart/form-data)
     - Phân tích metadata hình ảnh upload
     - Khớp với sản phẩm dựa trên đặc trưng trích xuất

PHẦN VII. HƯỚNG DẪN CÀI ĐẶT VÀ CHẠY DỰ ÁN
================================================================================

  Yêu cầu môi trường:
    - Node.js v14 trở lên (https://nodejs.org)
    - npm (đi kèm Node.js)
    - Trình duyệt hiện đại (Chrome, Edge, Firefox)

  Các bước thực hiện:

  Bước 1: Giải nén hoặc mở thư mục dự án
    > cd "WEB THU CUNG"

  Bước 2: Cài đặt các thư viện cần thiết
    > npm install

  Bước 3: Khởi động server (chế độ phát triển)
    > npm run dev
    [Kết quả mong đợi]: Server is running on https://kdkts.tools.edu.vn

  Bước 4: Mở ứng dụng trên trình duyệt
    Truy cập địa chỉ: https://kdkts.tools.edu.vn

  Lưu ý khi thử nghiệm tìm kiếm giọng nói:
    - Chỉ hoạt động trên HTTPS hoặc localhost
    - Cần cho phép quyền truy cập Microphone khi trình duyệt yêu cầu
    - Hỗ trợ tốt nhất trên Google Chrome
